About Facts & Hearts

Facts & Hearts combines fun trivia challenges with meaningful heartfelt questions. It’s designed to stimulate your mind and deepen your connections all in one place.


The purpose of this website is to provide a simple and engaging space where users can both learn new information and think deeply about meaningful topics. It is designed to be user-friendly and easy to navigate, making it accessible for different types of users. The system was developed using HTML, CSS, and Angular, and it integrates the Open Trivia Database API and Heartfelts Questions API to provide dynamic trivia questions.


This project was created as part of a web development requirement, aiming to demonstrate skills in building a responsive and interactive website. Overall, Facts & Hearts aims to offer a balanced experience that supports both knowledge improvement and personal reflection in a simple and enjoyable way.
